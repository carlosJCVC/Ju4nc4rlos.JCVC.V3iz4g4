## Description

[Mobile app](https://github.com/carlosJCVC/student-library-app)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Endpoints

- C and R : GET - POST {APP_URL}/students
- U and D : {APP_URL}/students/:id
